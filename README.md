# Car Service Tracker Demo

Static demo UI for a Car Service Tracker system built with Tailwind CSS.

This project is designed for presentation purposes and can be deployed directly on GitHub Pages without any backend or build tools.

---

## Features

- Modern dashboard UI
- Customer management page
- Vehicle management page
- Service tracking page
- CSV import demo page
- Responsive layout
- Tailwind CSS design
- Simple page switching with vanilla JavaScript
- Watermark support for demo/public showcase

---

## Tech Stack

- HTML5
- Tailwind CSS (CDN)
- Vanilla JavaScript

## Deploy to GitHub Pages

### Create a separate branch

```bash
git checkout -b gh-pages
```

### Push files

```bash
git add .
git commit -m "Add static demo"
git push origin gh-pages
```

### Enable GitHub Pages

Go to:

```txt
Repository → Settings → Pages
```

Then select:

```txt
Deploy from a branch
Branch: gh-pages
Folder: /root
```
